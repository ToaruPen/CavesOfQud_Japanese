# Localization Targets / Coverage

- `Docs/targets/*.md` には **UI パイプライン単位の対応状況**を整理する。
- `Docs/localization_targets.md` がファイル種別の俯瞰を担い、このフォルダでは **「どの画面のどの文字列が翻訳済みか」** を粒度高く記録する。
- 最低限の列例: `ContextID`, `Subsystem`, `UI`, `Status (NotStarted / Hooked / Verified)`, `Notes`.
- Phase5 のバックログに落とす要素もここから抽出する。
